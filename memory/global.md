## Never Ship
- Placeholder content in production
- Broken mobile layouts
- Pages without meta descriptions
- Forms without validation
- Images without alt text
- Hardcoded hex colors
- console.log in production
- Secrets in repo
- Gradients in illustrations (UI gradients are allowed)
- Claims without evidence (see: standards/evidence-based-coding.md)
- Commits with wrong git author email (must be tangison@proton.me — Vercel blocks deployments from unrecognized emails)

## Commit Format
[type]: [what changed] — Tangison Studio

## Design Defaults
- Typography: Cabinet Grotesk (display) + Satoshi (body) + JetBrains Mono (metadata) — never Inter, Roboto, Arial, system-ui
- Colors: Skeleton Bone, Atlantic Black, Signal Teal — one sharp accent, committed palette
- Spacing: CSS variables for every token
- Direction: Editorial, Namibian, flat-illustration — not generic AI-aesthetic
- Motion: cubic-bezier(0.16, 1, 0.3, 1), spring-physics navigation, staggered entrances
- Radius: 0px globally (Studio design language — max 4px if needed)

## SADC Defaults
- Mobile-first: 375px base
- Offline-aware: flag features requiring stable internet
- Low-bandwidth: WebP, compressed assets, minimal JS
- Payments: PayToday, DPO Pay, MTC Mobile Money (Namibia); M-Pesa (SADC)
- Communication: WhatsApp preferred for client-facing features
